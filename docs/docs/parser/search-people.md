# Поиск людей

Эта функция позволяет собирать информацию о людях из глобального поиска с учетом заданных критериев, преодолевая ограничение в 1000 пользователей. Результаты сохраняются в `(Папка проекта)\(SearchUsers)`, а имя файла формируется на основе текущей даты и времени на вашем компьютере.

Функция работает в комбинированном режиме. Сначала программа определяет количество доступных пользователей в поиске с использованием вашей учетной записи. Это необходимо для того, чтобы программа сгенерировала оптимальное количество запросов для обхода ограничения в 1000 человек (подробнее в разделе [Обход лимита VK в 1000 пользователей](#vk-1000)). 

Если на этом этапе становится ясно, что программа не может преодолеть лимит, или если количество результатов по выбранным критериям не превышает тысячу, программа сохраняет собранных пользователей, и задача считается завершенной.

В случае, если программа может обойти ограничение VK в 1000 пользователей, она генерирует необходимые запросы для получения как можно большего количества данных и переключается в режим мультиаккаунта.

??? question "Какие данные о пользователях собирает функция?"
    Это зависит от ваших настроек. Ознакомьтесь с разделом [Настройка Парсера](./index.md#settings).
